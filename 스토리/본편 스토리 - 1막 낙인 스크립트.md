개발 프로세스에서 바로 데이터셋(JSON, XML 또는 스크립트 딕셔너리)으로 파싱하여 관리할 수 있도록, 1막의 모든 출력 텍스트, 분기 옵션, 연동 시스템 이벤트를 구조화한 [1막: 낙인(Stigma) 메인 시나리오 스크립트 마스터 명세서]입니다.

실제 하드코어 텍스트 RPG 개발에 필요한 시스템 트리거 값, 가중치 변수, 아이템 ID 포인터를 명시하여 가독성과 개발 편의성을 극대화했습니다.

# 📝 1막: 낙인 (Protocol: Stigma) 스크립트 마스터 명세서

## 📌 [공통 메타데이터 & 초기 환경 설정]

* **구역 플래그**: `[City_Gate = 0 / Wasteland_Gate = 1]` (데드존 환경 적용)
* **평판 가중치($F$)**: 데드존 커뮤니티 평판 수치($R$) 기반 동기화 가동
* **자원 초기치**: `HP: 1,500 / 1,500`, `Hunger: 100 / 100`, `Thirst: 100 / 100`

### 코드 ID: `ACT1_LOOP_CORE`

* **이동 및 턴 소모 공용 스크립트 연산식**:

$$\text{Hunger}_{\text{turn}} = \text{Hunger}_{\text{current}} + \lfloor 5 \times (1 - R_{\text{decay}}) \rfloor$$


$$\text{Thirst}_{\text{turn}} = \text{Thirst}_{\text{current}} + \lfloor 6 \times (1 - R_{\text{decay}}) \rfloor$$
