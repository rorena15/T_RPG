# PROTOCOL: STIGMA - 1막: 낙인

> **PROTOCOL: STIGMA**는 시스템에 의해 폐기된 불량 코드로서 데드존에서 생존하고, 방공호 벙커를 장악하기까지의 여정을 그리는 **텍스트 기반 사이버펑크 RPG 게임**입니다. Windows CMD의 렌더링 문제를 극복하고 Rich 라이브러리를 통해 세련된 터미널 UI를 제공합니다.

## 요구사항

- **Python 3.8+**
    
- **필수 라이브러리:**
    
    - `colorama`
        
    - `rich`
        

## 설치 및 실행

### 1. 라이브러리 설치

터미널에서 아래 명령어를 입력하여 필요한 라이브러리를 설치합니다.

Bash

```
pip install colorama rich
```

### 2. 데이터베이스 초기화 및 실행

첫 실행 시 SQLite 데이터베이스가 자동으로 생성됩니다. `개발` 디렉토리로 이동한 뒤 게임을 실행하세요.

Bash

```
cd 개발
python Main.py
```

## 게임 플레이

### 게임 화면 구성

- **상태창:** 체력, 허기, 갈증, 장착 장비, 진행 턴 실시간 표시
    
- **그리드 맵:** 5x5 사이즈의 데드존 섹터 시각화
    
- **명령 메뉴:** 이동, 탐색, 인벤토리 관리를 위한 직관적인 메뉴 제공
    

### 주요 조작 가이드

|**키**|**기능**|**상세 설명**|
|---|---|---|
|**W / A / S / D**|그리드 이동|5x5 맵을 이동 (이동 시 허기 및 갈증 소모)|
|**F**|현재 위치 탐색|전투 돌입, 무작위 이벤트 발생, 자원 수집|
|**I**|인벤토리 관리|장비 장착, 소모품 사용, 불필요한 장비 분해|
|**J**|항법 일지 열람|지금까지 지나온 선택지와 스토리 기록 확인|
|**C**|현재 상태 저장|진행 상황을 세이브 파일로 저장|
|**Q**|게임 종료|게임을 종료 (종료 전 저장 여부 선택 가능)|

## 게임 시스템

### 생존 메커니즘

- **체력 (HP):** 시작 수치는 `1500`이며, `0` 이하로 떨어지면 **GAME OVER** 처리됩니다.
    
- **허기 / 갈증:** 매 턴마다 일정량 소모되며, 어느 하나라도 `0`에 도달하면 턴당 체력이 감소합니다.
    
- **자원 관리:** 탐색 및 분해로 얻은 **고철**을 모아 소모품을 구매하거나 상위 등급의 장비를 제작할 수 있습니다.
    

### 전투 시스템

- **주무기 공격:** 패널티 없는 가장 기본적인 공격입니다.
    
- **급조 바리케이드:** 방어 태세를 갖추어 피격을 반감시키고, 적의 분석 지수를 차감합니다.
    
- **패킷 우회 교란:** RAM을 `2` 소모하여 적을 해킹하고, 누적된 적의 분석 수치를 초기화합니다.
    
- **전술적 후퇴:** 전투에서 도망칩니다. 성공 확률은 플레이어의 **민첩 스탯**에 의존합니다.
    
- **소모품 사용:** 턴을 소모하여 인벤토리에 보유 중인 회복 아이템을 사용합니다.
    

### 진행 턴 기반 적 스케일링

턴이 지날수록 데드존의 적들이 점차 강화됩니다. 선택한 난이도에 따라 턴당 강화율이 다르게 적용됩니다.

- **이지 (Easy):** 턴당 `+1%` 강화
    
- **노멀 (Normal):** 턴당 `+2%` 강화
    
- **하드 (Hard):** 턴당 `+3.5%` 강화
    
- _※ 플레이어가 저체력 상태일 경우, 급격한 데드신을 방지하기 위해 밸런스 완화 보정이 적용됩니다._
    

### 선택 기반 스토리

- 주요 분기점마다 **3가지 선택지**가 제시됩니다.
    
- 선택에 따라 행동 코드 스탯인 **Kinetic**, **Scrap**, **Cyber** 수치가 누적됩니다.
    
- 가장 많이 누적된 행동 코드 유형에 따라 최종 엔딩 분기가 결정됩니다.
    

### 장비 및 퀘스트 시스템

- **11개 슬롯 부위:** 세분화된 장비 장착 시스템을 지원합니다.
    
- **5개 티어 등급:** `T0 유물`부터 `T4 급조` 등급까지 존재하며, 장비 성능(Power)에 비례해 캐릭터 스탯이 상승합니다.
    
- **장비 분해:** 필요 없는 장비를 분해하여 핵심 자원인 고철을 추출할 수 있습니다.
    
- **돌발 퀘스트:** 턴 진행 중 확률적으로 서브 미션이 발생하며, 기한 내 목표 달성 시 강력한 보상을 얻지만 실패 시 불이익을 받습니다.
    

## 터미널 실행 환경 추천

- **Windows Terminal (가장 권장)**
    
    - Microsoft Store에서 무료 다운로드 가능
        
    - Rich 라이브러리의 이모지, 한글 폰트 및 고해상도 색상 렌더링에 가장 최적화됨
        
- **VS Code 내장 터미널 (차선책)**
    
    - `Ctrl + `` 단축키로 터미널을 열어 실행
        
    - 추천 가독성 폰트: `D2Coding` 또는 `Courier New`
        
- **PowerShell 7+ (호환 가능)**
    
    - Windows Terminal 또는 VS Code와 연동하여 안정적인 구동 가능
        

## 파일 구조

Plaintext

```
개발/
├── Main.py              # 게임의 메인 루프 및 Rich UI 엔진 담당
├── db_init.py           # SQLite 데이터베이스 구조 생성 및 초기화 스크립트
├── sys_log.py           # 게임 내부 시스템 로깅 및 에러 추적
├── database.json        # 핵심 서사 텍스트, 소모품 스펙, 이벤트 데이터 정의
├── master_formulas.json # 데미지, 스케일링, 확률 계산용 밸런스 수식
├── stigma_data.db       # 로컬 스토리지를 위한 SQLite DB (실행 시 자동 생성)
└── stigma_save.json     # 플레이어 데이터 백업용 세이브 파일 (저장 시 자동 생성)
```

## 저장 및 로드

- **데이터 저장:** 게임 플레이 중 `C` 키를 누르면 현재 디렉토리에 `stigma_save.json` 파일이 실시간으로 생성되거나 갱신됩니다.
    
- **데이터 로드:** 메인 타이틀 화면에서 `2번(게임 불러오기)`을 입력하면 이전 저장 시점부터 이어서 플레이할 수 있습니다.
    

## 게임 진행 가이드

- **최종 목표:** 데드존 섹터를 안전하게 탐색하며 장비를 갖춘 뒤, 방공호 벙커 위치(4, 4)에 도달하여 최종 보스인 '스캐브 컬렉터'를 격파해야 합니다.
    
- **NPC 행상인 조우:** 탐색 도중 일정 확률로 상인을 만날 수 있습니다. 수집한 고철로 생존에 필요한 핵심 회복 소모품을 미리 구비하세요.
    
- **세계관 해금:** 장비 등급이 높아질수록 아이템에 숨겨진 딥 러닝 서사와 세계관 플레이버 텍스트가 잠금 해제됩니다.
