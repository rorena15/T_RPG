# PROTOCOL: STIGMA — 1막: 낙인

> **PROTOCOL: STIGMA**는 시스템에 의해 폐기된 불량 코드로서 데드존에서 생존하고, 방공호 벙커를 장악하기까지의 여정을 그리는 **텍스트 기반 사이버펑크 RPG**입니다.

> 선택이 당신을 만들고, 삶이 당신을 만든다.

![T_RPG Banner](assets/banner.png)

---

## 요구사항

- **Python 3.10+**
- **필수 라이브러리:**
  - `colorama` — 터미널 색상 출력
  - `pygame-ce` — 사운드 시스템 (BGM / 효과음)

```bash
pip install colorama pygame-ce
```

## 설치 및 실행

**EXE 실행 (권장)**
1. [릴리즈 페이지](../../releases/latest)에서 `Protocol_Stigma.exe` 다운로드
2. 압축 해제 후 `Protocol_Stigma.exe` 실행
3. Windows Defender 경고 시 → `추가 정보` → `실행` 선택

**소스 직접 실행**

```bash
pip install colorama pygame-ce
cd 개발
python Main.py
```

첫 실행 시 SQLite 데이터베이스(`stigma_data.db`)가 자동 생성됩니다.

**추천 터미널:** Windows Terminal (MS Store 무료 다운로드) → 한글 폰트 및 색상 렌더링 최적화  
차선책: VS Code 내장 터미널 (`D2Coding` / `Courier New` 폰트 권장)

## 게임 시스템

### 조작 가이드

| 키 | 기능 | 설명 |
|---|---|---|
| **W / A / S / D** | 그리드 이동 | 5×5 맵 이동 — 이동마다 허기·갈증 소모 |
| **F** | 현재 위치 탐색 | 전투 돌입 · 무작위 이벤트 · 자원 수집 (타일 수색 한도 적용) |
| **I** | 인벤토리 | 장비 장착, 소모품 사용, 장비 분해 |
| **J** | 항법 일지 | 플레이어 선택 기록 열람 |
| **C** | 저장 | `stigma_save.json`에 세이브 |
| **Q** | 종료 | 저장 여부 선택 후 종료 |

---

### 생존 메커니즘

- **HP:** 기본 1,500 (스탯 기반 상승). 0 이하 → GAME OVER
- **허기 / 갈증:** 매 턴 소모. 어느 하나라도 0에 도달하면 턴당 HP 감소
- **고철 (Materials):** 탐색·분해로 수집. 소모품 구매 및 스킬 발동 비용
- **Alert Level (0~100):** 적 세력의 경계 지수. 전투 조우 시 상승, 매 턴 -3 자동 감쇄. 높을수록 전투 조우 확률 증가 (SAFE 0–39 / CAUTION 40–69 / DANGER 70–100)
- **타일 수색 한도:** 타일마다 수색 가능 횟수가 랜덤 배정됨. 소진 후 쿨타임이 지나야 재수색 가능

---

### 전투 시스템

전투는 턴제로 진행됩니다.

| 키 | 행동 | 설명 |
|---|---|---|
| **1** | 주무기 공격 | 장비 Power 기반 데미지 — 치명타 확률 적용 |
| **2** | 급조 바리케이드 | 방어 태세 — 피격 반감 + 적 패턴 학습 차단 |
| **3** | 패킷 우회 교란 | RAM 2 소모 — 적 딥러닝 분석 수치 초기화 |
| **4** | 전술적 후퇴 | DEX 기반 도주 성공 확률 |
| **5** | 소모품 사용 | 인벤토리에서 회복 아이템 사용 |
| **6** | 보조 화기 | 네오 아크 AI 폐기 화기 발사 — 2회 제한, 소진 후 자동 파기 |
| **S / S1 / S2** | 스킬 사용 | 장착된 특화·보조 스킬 발동 |

**보스 딥러닝 시스템:** 보스는 전투 지속 시간에 따라 학습 지수(E)를 축적하며, 분기점마다 공격 패턴을 강화합니다. 특정 스킬로 학습 신호를 차단할 수 있습니다.

**진행 턴 기반 스케일링:**
- 이지: 턴당 +1% / 노멀: +2% / 하드: +3.5%
- 저체력 상태에서는 밸런스 완화 보정 적용

### 스킬 시스템 (v4)

스킬은 **특수 / 특화 / 보조** 3계층으로 구성됩니다.

#### 특화 스킬 (Special) — 각성 시 직업별 2개 자동 지급

| 직업 | 스킬 | 비용 | 효과 |
|---|---|---|---|
| 컴뱃 포스 | 유압 분쇄 | HP 15% | 다음 공격 ×1.8 + 방어력 50% 관통 |
| 컴뱃 포스 | 철제 의체 | 고철 10 | 3회 피격 피해 -40%~60% (VIT 기반) |
| 메카니컬 테크 | 고철 구조체 | 고철 20 | 2턴 피해 -25% + 보스 패턴 학습 차단 |
| 메카니컬 테크 | 과부하 수리 | 고철 10 | HP 즉시 회복 (INT 기반) + RAM +1 |
| 넷 포스 | 데이터 사이펀 | RAM 2 | 이번 전투 적 ATK -30% + 매 턴 RAM +1 회수 |
| 넷 포스 | 고스트 프로토콜 | RAM 1 | 이번 턴 반격 차단 + 다음 공격 치명타율 +30% |
| 황금 분할 | 종합 구동 | HP 5% + RAM 1 | VIT·INT·DEX 합산 즉각 피해 + 생존 수치 소모 |
| 황금 분할 | 균형점 | 고철 10 | HP·허기·갈증 각 +10% 동시 회복 |

슬롯: 최대 2개 장착 (각성 지급 2개가 기본값)

#### 특수 스킬 (Unique) — 별도 조건 달성 시 언락

| 스킬 | 언락 조건 |
|---|---|
| 오버클록 | 컴뱃 포스 — kinetic 가중치 누적 5+ |
| 생체 수확 | 넷 포스 — 연속 3전투 \|STR-INT\|≤3 유지 |
| 보초 인프라 | 메카니컬 테크 — 고철 구조체 3회 이상 사용 |
| 신호 추적 | 넷 포스 — cyber 가중치 누적 5+ |
| 그리드 침입 | 넷 포스 — 패킷 우회 5회 이상 |
| 프로토콜 글리치 | 황금 분할 — 특정 이벤트 연쇄 조건 |

#### 보조 스킬 (Aux) — 2막 이후 이벤트·퀘스트로 획득

| 스킬 | 효과 요약 |
|---|---|
| 바이탈 펌프 | 전투 시작 시 HP 회복 |
| 고철 방패 | RAM 소모 → 임시 방어력 부여 |
| 정크 캐논 | 고철 소모 → 즉각 데미지 |
| 바이오루프 | 치명타 발생 시 HP 소량 흡수 |
| 램 응축기 | 턴 종료 시 RAM 자동 +1 |
| 코드 컴파일 | 2턴간 보스 패턴 학습 차단 |
| 펄스 수류탄 | 범위형 데미지 + 보스 E 감소 |
| 키네틱 버스트 | 이번 턴 아웃고잉 피해 +15% |
| 신경 가속 | 다음 치명타 배율 2배 |
| 보이드 시프트 | 이번 턴 전투 이탈 100% 성공 |

슬롯: 최대 3개 장착

### 직업 시스템 (각성)

1막 클리어 후 **각성** 이벤트를 통해 직업이 결정됩니다.  
행동 가중치 (kinetic / scrap / cyber) 누적 비율에 따라 자동 배정됩니다.

| 직업 | 특성 | 배정 조건 |
|---|---|---|
| 컴뱃 포스 | 고화력 근접 전투 | kinetic 우세 |
| 메카니컬 테크 | 방어·수리·자원 활용 | scrap 우세 |
| 넷 포스 | RAM 해킹·디버프 | cyber 우세 |
| [검열 됨] | [검열 됨] | [검열 됨] 분배 |

### 장비 시스템

- **5등급:** T0 유물 · T1 고급 · T2 표준 · T3 정제 · T4 급조
- **11개 슬롯:** 주무기 / 사이버덱 / 의체부품 / 등 장비 / 얼굴 / 상의 / 하의 / 신발 / 목걸이 / 반지 / 특화부품
- **장비 분해:** 불필요 장비 → 고철 추출
- **세계관 플레이버:** 장비 등급이 높아질수록 숨겨진 서사 잠금 해제

---

### 돌발 퀘스트 및 이벤트

- **돌발 퀘스트:** 턴 진행 중 12% 확률 발생. 기한 내 목표 달성 시 강력한 보상 지급
- **랜덤 이벤트:** 탐색 시 무작위 발생 (단순 결과형 / 선택지형 / 특수 아이템 획득형)
- **행상인 조우:** 고철로 소모품 구매 가능

---

### 선택 기반 스토리

- 주요 분기점마다 선택지 제공 → kinetic / scrap / cyber
- **3가지 기본 엔딩** + 특정 조건을 만족해야 열리는 **히든 엔딩**
- 1막 클리어 후 **2막: 파벌 선택** 티저 공개

## 사운드

`sound.py`가 pygame-ce를 통해 BGM과 효과음을 관리합니다.

| 채널 | 재생 조건 |
|---|---|
| 맵 환경음 | 데드존 탐색 중 루프 재생 |
| 서사 BGM | 스크립트·이벤트 진행 중 |
| 전투 BGM | 전투 돌입 시 자동 전환 |
| 심장박동 경보 | 허기 또는 갈증 ≤ 10 진입 시 활성 |

사운드 디바이스가 없는 환경에서는 자동으로 무음 모드로 폴백됩니다.

## 저장 / 로드

- **저장:** 게임 중 `C` 키 → `stigma_save.json` 자동 생성·갱신
- **로드:** 타이틀 화면 `2번(게임 불러오기)` → 이전 저장 시점부터 재개
- **항법 일지:** `J` 키로 지금까지의 선택 기록 열람
