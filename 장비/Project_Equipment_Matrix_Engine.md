본 문서는 게임 내 존재하는 모든 장비의 위력, 강화 확률, 재료 부스팅 및 페널티의 동적 변화를 제어하는 마스터 연산 코어 엔진 명세서입니다.

## 1. 장비 제어 변수 선언 및 배열 정의

모든 장비 자산은 등급 지수($T$)와 슬롯 구조 지수($S$)의 행렬 좌표 $[T][S]$에 의해 베이스 값이 동적으로 결정됩니다.

###  등급 변수 매핑 ($T$)

- $T = 4$ : 4등급 급조 (Scrap) ➔ $P_{base} = 20$, $C_{bio} = 0$, $C_{ram} = 0$
    
- $T = 3$ : 3등급 규격 (Standard) ➔ $P_{base} = 50$, $C_{bio} = 1$, $C_{ram} = 1$
    
- $T = 2$ : 2등급 정제 (Refined) ➔ $P_{base} = 95$, $C_{bio} = 2$, $C_{ram} = 2$
    
- $T = 1$ : 1등급 기업제 (Corporate) ➔ $P_{base} = 160$, $C_{bio} = 4$, $C_{ram} = 3$
    
- $T = 0$ : 0등급 유물 (Legacy) ➔ $P_{base} = 280$, $C_{bio} = 8$, $C_{ram} = 6$
    

###  슬롯 가중치 변수 매핑 ($W_{part}$)

- $S_1 (\text{주무기}), S_2 (\text{사이버덱}) \rightarrow \mathbf{1.5}$
    
- $S_3 (\text{의체부품}), $S_4 (\text{등장비}) \rightarrow \mathbf{1.2}$
    
- $S_5 \sim S_8 (\text{얼굴/상의/하의/신발 외장}) \rightarrow \mathbf{1.0}$
    
- $S_{9} \sim S_{11} (\text{목걸이/반지/특화부품}) \rightarrow \mathbf{0.5}$
    

##  2. 코어 연산 공식 세션 (Core Formula)

### ① 최종 강화 장비 위력 산출식 ($EquipPower$)

플레이어가 장비를 강화($k$)할 때 상승하는 위력의 복합 스케일링 함수입니다 ($0 \le k \le 10$).

$$EquipPower = \lfloor P_{base} + \Delta P(k) \rfloor \times \text{Rebuild\_Bonus}$$

- **강화 단계별 추가 위력 변수 ($\Delta P(k)$) 가산 행렬:**
    
    - $k = 0$ : $+0$
        
    - $1 \le k \le 3$ : $k \times 10$
        
    - $4 \le k \le 6$ : $30 + (k - 3) \times 25$
        
    - $7 \le k \le 9$ : $105 + (k - 6) \times 50$
        
    - $k = 10$ : $+350$ (최고 임계 스택 도달 보너스 포함)
        
- **리빌딩 보너스 상향 승수 ($\text{Rebuild\_Bonus}$):**
    
    - 일반 드롭 및 제작 무기 장착 시: $1.0$
        
    - 1등급 +10강 스택을 소모해 0등급 유물로 승급 리빌딩 완수한 장비인 경우: **$1.15$ (상시 15% 위력 버스팅)**
        

### ② 실시간 누적 결속 페널티 공식 ($Cost_{total}$)

강화 단계가 임계점을 넘을 시 $M_{cost}$ 가중치가 적용되어 생존 지표를 압박하는 자원 제한 식입니다.

$$\text{Bio\_Cost} = C_{bio} \times W_{part} \times M_{cost}(k)$$

$$\text{RAM\_Lock} = C_{ram} \times W_{part} \times M_{cost}(k)$$

- **강화 수치별 페널티 증가 가중치 행렬 ($M_{cost}(k)$):**
    
    - $0 \le k \le 3$ : $\times 1.0$
        
    - $4 \le k \le 6$ : $\times 1.2$
        
    - $7 \le k \le 9$ : $\times 1.5$
        
    - $k = 10$ : $\times 2.0$
        
- **🚨 4등급(급조) 특약 예외 조항:** 급조 장비($T=4$)는 기본 페널티가 $0$이지만, 마스터 단계($k=10$) 도달 시에 한해 예외적으로 $C_{bio}=2, C_{ram}=1$의 고정 상수가 강제 주입 연산됩니다.
    

### ③ 🎲 파괴 없는 동적 강화 성공 확률 공식 ($Pr_{total}$)

실패 스택과 맵에서 채굴한 원자재의 부스팅 효율을 완벽히 흡수하는 제어식입니다.

$$Pr_{total} = \min(1.0, Pr_{base}(k) + \sum Pr_{pity} + Pr_{material})$$

- **강화 단계별 성공/천장 수치 매트릭스:**
    
    - $+1 \sim +3$ : $Pr_{base} = 0.80$, $Pr_{pity} = +0.10$
        
    - $+4 \sim +6$ : $Pr_{base} = 0.50$, $Pr_{pity} = +0.15$
        
    - $+7 \sim +9$ : $Pr_{base} = 0.25$, $Pr_{pity} = +0.20$
        
    - $+10$ 마스터 : $Pr_{base} = 0.10$, $Pr_{pity} = +0.25$
        
- **재료 부스팅 확률 가산 변수 ($Pr_{material}$):**
    
    - `구시대 티타늄 고철` : 개당 $+0.10$ (최대 3개 제한, $+0.30$)
        
    - `정제 코어 플루이드` : 개당 $+0.25$ (최대 2개 제한, $+0.50$)
        

## ⚔️ 3. 최종 연동 대미지 인코딩 수식 ($Display \, DMG$)

최종적으로 연산 엔진이 대미지를 인코딩하여 화면에 로그를 뿌릴 때, 위의 변수 엔진에서 도출된 $EquipPower$를 주 공식에 직결 대입합니다.

$$\text{DMG} = \max\Big(1, \lfloor (W \times \mathbf{EquipPower}) \times F \times P \rfloor - DEF_{eff}\Big)$$

$$Display \, DMG = \text{DMG} \times 100,000 \quad \text{[최종 십만 배수 한글 포맷 렌더링]}$$

- **변수 정의:**
    
    - $W$: 유저 고유 스킬 자체 위력 계수
        
    - $F$: 평판 지수 제로섬 가중치 ($1 + R/2000 \times \text{지역 플래그}$)
        
    - $P$: 패턴 노출도에 따른 패널티 함수 ($E > 20$ 일 때 최하 고정치 $0.5$)
        
    - $DEF_{eff}$: 관통과 디버프가 최종 정합된 적의 유효 방어력
        

## 1. 🧪 강화 재료의 소모량 및 획득 공식 누락 (Material Cost & Economy)

확률 부스팅 변수($Pr_{material}$)와 천장 규칙은 선언되었으나, **1회 강화 시도 시 소모되는 재료의 정량적 개수**와 **기계 괴수 해체 시 드롭되는 재료 연산식**이 누락되어 자원 인플레이션 통제가 불가능한 상태입니다.

### 📐 A. 강화 단계별 재료 소모 수식 ($Req_{count}$)

강화 시도 시 소모되는 재료의 개수는 단계($k$)의 제곱 팩터를 추종합니다.

$$\text{소모량} = \max\Big(1, \lfloor \alpha \times (k + 1)^{1.2} \rfloor\Big)$$

- **4등급(급조) 내지 3등급(규격):** 일반 고철 ($\alpha = 1.0$) 소모.
    
- **2등급(정제) 내지 1등급(기업제):** 티타늄 고철 ($\alpha = 1.5$) + `정제 코어 플루이드` 상수가 $k \ge 7$ 구간부터 회당 1개씩 강제 추가 결속.
    

### 📐 B. 필드 및 괴수 해체 시 재료 적출 공식

민첩 스탯($A_{DEX}$)과 사체 해체 정밀도가 재료의 획득량($Drop$)을 지배합니다.

$$\text{Drop}_{\text{Scrap}} = \max\Big(1, \lfloor f(A_{DEX}) \times \text{괴수 체급 상수} \rfloor\Big)$$

- 유저가 민첩 효율 임계점인 15 및 25 구간을 돌파할 때마다 고가치 재료인 `정제 코어 플루이드`가 드롭 테이블 플래그에 가산(`+1`) 파싱되는 수학적 근거를 부여합니다.
    

## 2. 🛡️ 방어 및 저항력 스케일링 변수 누락 (DEF_eff vs HP Display)

대미지 공식의 끝단에 직결되는 적의 최종 유효 방어력($DEF_{eff}$) 수식은 정립되었으나, 반대로 **플레이어가 확장 장비 슬롯(상의, 하의, 신발, 얼굴)을 통해 얻는 최종 방어력과 생명력의 내부 연산 스케일 상수**가 누락되어 있습니다.

### 📐 플레이어 최종 실효 방어력 공식

$$\text{Player\_DEF}_{eff} = \sum_{i=5}^{8} \Big( Slot_{i} \, 고유 \, 위력 \times M_{cost}(k) \Big) \times f(A_{VIT})$$

- **맥락 정합성:** 체력($A_{VIT}$) 중심의 '전선 사수' 빌드가 외장 레이어 장비를 고강화($M_{cost} = 2.0$)했을 때, 표시 생명력(Display HP 백배수)과 시너지를 내며 수만 단위의 타격을 고작 1단위의 최소 정수 대미지로 리액션 로그를 띄우는 완전한 계산 근거가 완성됩니다.
    

## 3. ⚖️ 무기 외 장비(방어구/장신구) 강화 시 성능 증가 변수 누락

장비 위력($EquipPower$) 공식의 $\Delta P(k)$ 가산 행렬은 주무기와 사이버덱의 화력(십만 배수 대미지)을 기준으로 설계되어 있습니다. 방어구나 장신구 슬롯 장비를 강화했을 때 '위력 대신 무엇이 증가하는가'에 대한 속성 변수 변격 조항이 누락되었습니다.

### 🔄 슬롯 카테고리별 강화 증가 속성 변격 조항

- **방어 외장 레이어 ($S_5 \sim S_8$):** 강화 단계 $k$당 기초 위력이 오르는 대신, 적의 물리 방어 관통을 역쇄시키는 **고유 보루 저항력 상수($PEN_{\text{resist}}$)가 단계당 $+5$씩 정량 가산**됩니다.
    
- **모듈러 장신구 레이어 ($S_{9} \sim S_{10}$):** 강화 단계 $k$당 상태 이상 오염 스택의 **자연 소거값($Recovery$) 상수가 단계당 $+1.5$씩 누적 가산**되어, 하드코어 환경 타일에서의 생존력을 증명합니다.
    

## 🔍 최종 무결성 검증 (Symmetry Guard)

- **검증:** 새로 보완된 재료 소모 스퀘어 함수와 방어구 전용 강화 변격 조항은 기존의 11개 슬롯 가중치 및 이원화 표시 스케일링 체계와 정확한 대칭을 이룹니다. 당연한 상식 선에서 연산 흐름을 방해하거나 설정이 깨지는 사각지대가 존재하지 않으므로, 억지 비판 없이 완벽히 정합합니다.
